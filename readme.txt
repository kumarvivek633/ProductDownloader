Steps to run the Project

1) Create db schema using the scripts provided inside ProductDownloader dir.
2) To build the backend code run 'mvn clean install' from  ProductDownloader/pd-api
3) once the build will be succesfull then run the jar from ProductDownloader/pd-api/application/target using command java -jar pd.jar.
4) To run the ui the prerequisit is to have Node 7+ installed. after the hit npm install from ProductDownloader/pd-ui directory.
5) after it is successfull hit ng server from ProductDownloader/pd-ui directory.
6) Hit the url https://localhost:4200 and use credentials as vkumar and pass.
